About EggPi

EggPi is a DIY file sharing system based on RasperryPi, Node.js, PSQL, openWRT, and open source technologies. EggPi is designed to allow users to easily and securely share and stream files across a secure wireless LAN. Users share communal permisions to upload, download, and stream files. The SysOp is authorized to edit hardware and software configurations, while an admin account is required to delete media from communal uploads.

An Arduino LCD sheild is used to provide user feedback to SysOp. The user login event and the associated user email is graphically displayed for instant network monotoring.

EggPi utilizes UPnP protocol in order to stream content from our miniDLNA server to any UPnP client. Our recomended UPnP media player is VLC. Accessing EggPi is as easy as connecting to the EGGNOGG server under the Local Network options.
